# OAUTH2-github-API
### Clone repository
```
$ git clone https://github.com/bariss48/OAUTH2-Github-API.git
```
### cd /server 
```
$ cd server (in root directory)
```
### install dependencies
```
$ yarn install
```
#### you must configure your github client ID and github client Secret.

### then you can run the server
```
$ yarn dev start
```
